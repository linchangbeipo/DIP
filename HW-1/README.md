# HW1 - Image Warping

## Installation
To install requirements:  
`python -m pip install -r requirements.txt `

## Running
To run **basic transformation**, run:  
`python run_global_transform.py `

To run **transformation guided by points**, run:  
`python run_point_transform.py`
